欧拉函数：任意给定正整数n，请问在小于等于n的正整数之中，有多少个与n构成互质关系？计算这个值的方法就叫做欧拉函数，以φ(n)表示。如果n可以分解成两个互质的整数之积，即 n = p * k ，则φ(n) = φ(p * k) = φ(p1)*φ(p2) 【n是质数的某一个次方，p为质数，k为大于等于1的整数】 欧拉定理：如果两个正整数m和n互质，那么m的φ(n)次方减去1，可以被n整除。 费马小定理：欧拉定理的特殊情况，如果两个正整数m和n互质，而且n为质数！那么φ(n)结果就是n-1。 模反元素：还剩下最后一个概念，模反元素：如果两个正整数e和x互质，那么一定可以找到整数d，使得 ed-1 被x整除，或者说ed被x除的余数是1。那么d就是e相对于x的模反元素。

Created2024-05-12
当使用kill -n命令时，-n后面的数字代表着要发送的特定信号。以下是一些常见的kill -n信号及其对应的作用： SIGHUP (1) - 终端挂起或断开时发送的信号。 SIGINT (2) - 中断进程，通常由键盘输入 Ctrl + C 发送。 SIGQUIT (3) - 退出进程，通常由键盘输入 Ctrl + \ 发送。 SIGILL (4) - 非法指令信号，指示进程执行了非法、未定义或未实现的指令。 SIGKILL (9) - 强制终止进程，不允许进程做清理或善后工作。 SIGTERM (15) - 正常终止进程，允许进程做清理操作。 SIGUSR1 (10) - 用户自定义信号1。 SIGUSR2 (12) - 用户自定义信号2。 SIGSEGV (11) - 段错误，通常指示进程访问了无法访问的内存区域。 SIGPIPE (13) - 管道破裂，用于在读取已关闭的管道时发送给进程。 SIGALRM (14) - 由定时器发送的信号。 这些信号允许用户以不同的方式控制进程，从正常终止到强制终止等，以及一些特定的系统事件或用户定义的信号。
